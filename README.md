# NB System — Natural Balance Club

Cash flow management suite for Natural Balance Club (Mexico). Single-page web apps on GitHub Pages backed by Google Apps Script + Google Sheets.

**Live:** [louieelizondo.github.io/nb-app](https://louieelizondo.github.io/nb-app/)

## Apps

| App | Purpose |
|-----|---------|
| Dashboard | Monthly KPIs, charts, financial overview |
| Portal de Pagos | Supplier invoice management |
| Captura de Facturas | AI-powered invoice capture (camera → Claude vision) |
| Corte de Caja | Daily cash register reconciliation |
| Registro de Ingresos | Daily income, Sobre 2, invoicing |
| Inventario | Product catalog, counting, orders |

## Tech Stack

- **Frontend:** Static HTML/CSS/JS (GitHub Pages)
- **Backend:** Google Apps Script
- **Database:** Google Sheets
- **Auth:** Google OAuth2
